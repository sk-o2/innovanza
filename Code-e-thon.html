What is Code-E-Thon?

Code-e-thon is designed in a way so that all the aspects of coding are covered. It contains competitive coding problems. Everything is designed according to the participants’ point of view. Programming language is not a barrier. The competition will support the syntax of all programming languages. In the immersive world of competitive programming put your coding skills to test and be the winner.


Flow of Event

  • Before the main event a briefing will be provided to the students about the event, so that participants get to know well about the event.
  • After the briefing the registration form will be circulated.
  • Event will be only of a round which will run for 3 hours.
  • Participants will be provided with the system and the event will start.
  • After the round event will end and result will be declared later.

Rules and Regulations of the event

  • This event is open to 1st, 2nd and 3rd year students of any recognized institution or college.
  • Copying someone else’s code will lead to immediate disqualification from the whole event.
  • Discussing any problems (or any aspect of the problem) on any other platform on internet, if identified, could lead to disqualification of the participant from the contest.
  • Discipline should be maintained throughout the event.
  • Participants must coordinate properly with the event coordinators.
  • Asking anything related to the competition problems is not allowed during the contest.
